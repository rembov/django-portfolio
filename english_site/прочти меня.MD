## Что такое pip?

`pip` — это стандартный менеджер пакетов для Python. Он позволяет скачивать и устанавливать библиотеки из интернета одной командой. Например, когда ты писал `pip install django`, pip нашёл последнюю версию Django на официальном сайте Python Package Index (PyPI) и поставил её на твой компьютер.

## Как установить любой пакет через pip

1. **Открой командную строку (терминал):**
   - Если ты в PyCharm: внизу есть вкладка **Terminal** — нажми на неё.
   - Если просто в Windows: нажми `Win + R`, введи `cmd` и нажми Enter.

2. **Проверь, что pip работает:**
   ```bash
   pip --version
   ```
   Если видишь сообщение типа `pip 24.0 from C:\...` — всё ок. Если ошибка, значит, при установке Python не была поставлена галочка «Add Python to PATH». Тогда проще всего переустановить Python, обязательно отметив эту галочку.

3. **Установка пакета:**
   ```bash
   pip install название_пакета
   ```
   Например, чтобы установить библиотеку `requests`, введи:
   ```bash
   pip install requests
   ```
   Pip скачает и установит пакет. После успешной установки увидишь сообщение `Successfully installed ...`.

4. **Посмотреть установленные пакеты:**
   ```bash
   pip list
   ```
   Покажет список всех установленных библиотек.

5. **Удалить пакет:**
   ```bash
   pip uninstall название_пакета
   ```

## Полная инструкция по запуску проекта English Easy Peasy

Теперь дам пошаговую инструкцию — от получения проекта до его запуска. Делай всё по порядку.

### Шаг 1. Установка Python
1. Скачай Python с официального сайта: [python.org/downloads](https://www.python.org/downloads/)
2. Запусти установщик, в самом низу первого окна **обязательно поставь галочку** «Add Python to PATH».
3. Нажми «Install Now» и дождись окончания.
4. Проверь: открой командную строку и введи `python --version`. Должно показать версию Python.

### Шаг 2. Установка PyCharm (Community Edition)
1. Скачай PyCharm CE: [jetbrains.com/pycharm/download](https://www.jetbrains.com/pycharm/download/)
2. Установи, ничего не меняя (просто жми Next).

### Шаг 3. Получение проекта
Тебе должны были дать папку с проектом (обычно архив `english_site.zip`). Распакуй её на рабочий стол или в удобное место. Внутри должен быть файл `manage.py`.

### Шаг 4. Открытие проекта в PyCharm
1. Запусти PyCharm, выбери **Open** и укажи папку `english_site`.
2. Проект откроется. Внизу будет вкладка **Terminal** — открой её.

### Шаг 5. Установка Django через pip
В терминале PyCharm введи команду:
```bash
pip install django
```
Подожди, пока установка завершится.

### Шаг 6. Создание базы данных
В том же терминале выполни:
```bash
python manage.py migrate
```
После этой команды в папке проекта появится файл `db.sqlite3` — это база данных.

### Шаг 7. Загрузка начальных данных (слова и награды)
Проект уже содержит готовые наборы слов. Загрузи их командами (по очереди):
```bash
python manage.py loaddata initial_achievements
python manage.py loaddata initial_words
```

### Шаг 8. Создание администратора (для входа в админку)
Выполни:
```bash
python manage.py createsuperuser
```
Придумай логин, email (можно пропустить) и пароль (не менее 8 символов). Запомни их.

### Шаг 9. Запуск сервера
Введи команду:
```bash
python manage.py runserver
```
Ты увидишь сообщение, что сервер запущен по адресу `http://127.0.0.1:8000/`.

### Шаг 10. Открытие сайта в браузере
Открой браузер и перейди по адресу: `http://127.0.0.1:8000/`.  
- Если ты не залогинен, увидишь приветственную страницу с предложением войти или зарегистрироваться.
- После регистрации или входа станут доступны все разделы: Слова, Игра, Грамматика, Говори!, Награды.

Админка находится по адресу `http://127.0.0.1:8000/admin/` — там можно добавлять и редактировать слова.

### Как остановить сервер
Вернись в терминал PyCharm и нажми `Ctrl + C`. Сервер остановится. Чтобы запустить снова — повтори команду `python manage.py runserver`.

### Если что-то пошло не так
- **Ошибка «python не является командой»** — значит Python не добавлен в PATH. Переустанови Python с галочкой.
- **Ошибка «pip не найден»** — та же причина.
- **Ошибка при миграциях** — проверь, что ты находишься в папке с `manage.py`.
- **Не работает микрофон** — используй Chrome или Edge и разреши доступ к микрофону.
