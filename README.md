# 🐍 Портфолио Django-проектов

Привет! Это репозиторий с примерами моих работ на Django. Здесь собраны небольшие, но рабочие проекты - от лендингов и блогов до каталогов и учебных приложений.

---

## 📁 Проекты в этом портфолио

### 1. 📚 Сайт для изучения английского (`english_site`)
Небольшое веб-приложение с разделами для изучения слов, грамматики и простыми играми.
*   **Слова:** Таблицы с переводом.
*   **Грамматика:** Правила и упражнения на выбор формы глагола.
*   **Интерфейс:** Простой и понятный для пользователя.

### 2. 📦 Каталог книг, фильмов и игр (`katalog`)
Проект для каталогизации контента с фильтрацией по категориям и годам.
*   **Список элементов:** Отображение элементов каталога с рейтингом.
*   **Фильтрация:** Возможность фильтровать контент.
*   **Детальная страница:** Страница с подробным описанием (заглушка).
